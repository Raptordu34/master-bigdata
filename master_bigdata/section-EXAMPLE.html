Bloc multi-lignes — pre

-- Vue matérialisée avec index
CREATE MATERIALIZED VIEW ventes_mensuelles AS
    SELECT
        DATE_TRUNC('month', date_vente) AS mois,
        region,
        SUM(montant)                    AS total
    FROM ventes
    GROUP BY 1, 2
WITH DATA;

CREATE INDEX ON ventes_mensuelles (mois, region);

Démonstration / Preuve

Pour tout graphe connexe non orienté à n sommets, le nombre minimal d'arêtes est $n - 1$ (arbre couvrant).
Par récurrence sur n. Pour n = 1 : l'arbre à un sommet a 0 arête, soit $n - 1 = 0$. Supposons vrai pour tout graphe à k < n sommets. Soit G connexe à n sommets. En retirant une feuille v (sommet de degré 1), le graphe G' = G \ {v} reste connexe à n-1 sommets et possède, par hypothèse, au moins $n - 2$ arêtes. En rajoutant v et son arête, on obtient au moins $n - 1$ arêtes.

Algorithmes (pseudo-code)

Entrée : X (n points), k (nombre de clusters), ε (convergence)
Sortie : Affectation c[], centroïdes μ[]

1. Initialiser μ₁..μₖ aléatoirement parmi X
2. Répéter :
   a. Pour chaque xᵢ ∈ X :
         c[i] ← argmin_j ||xᵢ − μⱼ||²
   b. Pour chaque j = 1..k :
         μⱼ ← moyenne({ xᵢ | c[i] = j })
3. Jusqu'à ||Δμ|| < ε

Complexité : O(n · k · d · i)  — i = itérations, d = dimensions
